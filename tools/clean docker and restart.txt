clean docker and restart

1. List everything so you know what you’re deleting:

docker ps -a --format "table {{.ID}}\t{{.Names}}\t{{.Status}}\t{{.Image}}"


2.Stop & remove the old containers (ignore errors if some don’t exist):

docker rm -f llmscope_api llmscope_web llmscope_monitor frontend backend monitor `
  zealous_thompson youthful_curie unruffled_villani sharp_franklin


3. Bring your current stack fully down (and remove orphans):

docker compose down --remove-orphans


4. (Optional tidy) prune unused networks/containers (safe):

docker container prune -f
docker network prune   -f
